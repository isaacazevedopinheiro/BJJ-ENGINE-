# BJJ-ENGINE-

Projeto Individual de um site desenvolvido como uma iniciativa pessoal/profissional com foco no aprendizado, aplicando os conceitos abordados pela faculdade SPTech

A escolha deste projeto foi influenciada não por objetivos tecnicos mas por questões de experiencias e preferencias pessoais. 
-especificamente minha paixao pelo jiu-jitsu.

O Jiu-Jitsu é uma arte marcial japonesa,
conhecida como "arte suave",
focada em luta agarrada (grappling),
técnicas de alavanca,
imobilizações e finalizações

Fui apresentado ao jiujitsu em 2024, 
mas sempre tive experiencias com desportos de combate e artes marciais.


No jiujitsu aprendi a importancia de: 
disciplina,
tomada de decisao em tempo real,
adaptaçao constante,
analise de comportamento.
